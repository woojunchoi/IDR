### Setup Instructions

> 1. Fork this repo
> 2. Clone your forked copy of this repo
>    - `git clone https://github.com/[Your Username]/IDR.git`
> 3. Change directory into the newly cloned repo
>    - `cd HiltonAsessment`
> 4. Install dependencies 
>    - `npm install`
> 5. Start dev server
>    - `npm start`
> 6. Open a browser and navigate to `localhost:3000`
